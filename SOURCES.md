# Sources

This project prioritizes the provided workspace materials. External sources are
used only for formula, shape, and notation verification or for missing standard
definitions.

| Source | Location | Used for |
|---|---|---|
| Deep Learning Specialization C1 notes | `C1/` | Logistic regression, vectorization, shallow/deep networks, forward propagation, backpropagation |
| Deep Learning Specialization C2 notes | `C2/` | Train/dev/test split, bias/variance, regularization, dropout, batch normalization, mini-batch GD, Momentum, RMSProp, Adam, hyperparameter tuning |
